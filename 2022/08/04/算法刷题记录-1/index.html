  • 使用前提条件是数组有序且无重复

  • [35. 搜索插入位置 - 力扣（LeetCode）]:

  • int searchInsert(vector<int>& nums, int target) {
    int i=0;
    int j=nums.size()-1;
    while(i<=j)
    {
        int mid = (i+j)/2;
        if(target>nums[mid])
            i=mid+1;
        else if(target<nums[mid])
            j=mid-1;
        else
            return mid;
    }
    return i;//区别在于找不到需放在该放的位置
}

[19. 删除链表的倒数第 N 个结点 - 力扣（LeetCode）]:

快慢指针，快指针先走n步，然后快慢指针同时走直至快指针走到链表结尾i，此时慢指针指向删除节点前置结点，删除即可。

[面试题 02.07. 链表相交 - 力扣（LeetCode）]:

大致思路仍为快慢指针，首先计算两个链表的长度差，让长的链表先走这些差值，接下来两个链表指针一同前进，直至指向同一结点

[142. 环形链表 II - 力扣（LeetCode）]:

判断链表有环：快指针每次走两步，慢指针每次走一步，若有环，快慢指针必相遇。

找到成环的尾指针：从快慢指针的起始位置和相遇位置分别设置新指针同步率前进（走一步），当新的两指针相遇时，即为成环尾指针。

注意：代码第八行，必须用fast指针作为循环判断条件，以免空链表和只有一个元素的链表，若用slow作为判断条件，以下情况会造成空指针的next操作报错。

此时，fast->next->next会因为fast->next是空指针而无法next而报错
